

Variables:

offset: utc - local hour in minutes.
planetOfDay: array with planet sequence during week, starting Sunday.
hourSequenceOfDay: object of week days keys and array of planets in hour sequence as value.
fieldCity: page element that gets City name.
fieldPlanet: page element that gets Planet of day name.
fieldDate: page element that shows the selected date.
//location: array of string lat, lng.
lat: latitude string.
lng: longitude string.
city: name of city for location.
today: today date string (yyyy-mm-dd).

Functions:

dateObjToStr(dateObj): converts date format / input date object / output date string yyyy-mm-dd.
getLocation(): wrapper from callback to promise. get position from browser (using getCurrentPosition webAPI) / input none / output location object (coords.latitude and coord.longitude).
getCoord(): extracts latitude and longitude from helper/wrapper getLocation / input none / output array of strings [latitude, longitude].
getCity(lat, lng): get City name from location (using locationiq API) / input lat, lng string / output city name string.

calculatePlanet(): calculate planet of day / input date (string) / output planet (string).
